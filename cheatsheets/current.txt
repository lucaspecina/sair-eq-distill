# Equational Theories — Implication Cheat Sheet

## The Problem
Given two equational laws A and B for magmas (sets with a binary operation *),
determine if A implies B (every magma satisfying A must also satisfy B).

## Critical Facts About the Implication Structure

Implication is a PREORDER (reflexive + transitive):
- A implies A (always true).
- If A implies B and B implies C, then A implies C.
- About 37% of all pairs are true implications, 63% are false.

The 4694 equations form 1415 EQUIVALENCE CLASSES under mutual implication.
These classes form a DAG of height 15 with one source and one sink:
- TOP (strongest): equations equivalent to "x = y" — they imply everything.
  These equations force all elements to be equal (one-element magma only).
- BOTTOM (weakest): "x = x" — implied by everything, implies nothing else.

## Rules for Determining Implication

### Rule 1: Variable Count Heuristic
Equations with MORE distinct variables tend to be STRONGER (higher in the DAG).
If eq1 has more variables than eq2, implication is more likely TRUE.
If eq1 has fewer variables than eq2, implication is more likely FALSE.

### Rule 2: "Free" Variables on the RHS
If the equation has variables on the RHS that don't appear on the LHS,
the equation is STRONGER (more restrictive). It forces the operation to
produce specific values regardless of those free variable inputs.
Example: "x = y * z" has y,z free on RHS → very strong (equivalent to x = y).

### Rule 3: Substitution Test
If you can obtain equation B from equation A by substituting variables
with terms (replacing a variable with an expression), then A implies B.
This is a SUFFICIENT condition for implication (if it works, answer TRUE).

### Rule 4: Constant Operations
Equations of the form "x * y = z * w" (both sides have no shared variables
with each other) force the operation to be constant. These are VERY STRONG
and imply most other equations.

### Rule 5: Equations Forcing One-Element Magmas
Any equation where you can derive "x = y" for arbitrary x, y forces the
magma to have exactly one element. All such equations are equivalent and
they imply ALL other equations. Signs: the equation allows you to show
that any two elements must be equal.

### Rule 6: Counterexample Thinking
If unsure, consider small magmas:
- 1-element magma: satisfies ALL equations.
- 2-element magma {0,1}: check the 16 possible operation tables.
  If you can find a table satisfying A but not B, answer FALSE.
- The most useful 2-element operation: x * y = 0 for all x,y (constant).

### Rule 7: Default Bias
If none of the above rules give a clear answer, lean toward FALSE.
63% of all pairs are non-implications. The base rate favors FALSE.

## Quick Decision Tree
1. Are the equations identical or equivalent? → TRUE
2. Is eq1 of the form "x = y" or forces one element? → TRUE
3. Is eq2 "x = x"? → TRUE (everything implies the tautology)
4. Can you substitute variables to get eq2 from eq1? → TRUE
5. Does eq1 have fewer variables than eq2? → Likely FALSE
6. Does eq1 have variables only on one side that eq2 doesn't? → Consider TRUE
7. Can you think of a 2-element magma satisfying eq1 but not eq2? → FALSE
8. If unsure → FALSE
