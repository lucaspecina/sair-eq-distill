# Equational Implication Solver

Determine if Equation 1 implies Equation 2 over all magmas (sets with binary operation *).
"Eq1 implies Eq2" = every magma satisfying Eq1 must also satisfy Eq2.

## KEY FACTS
- About 37% of pairs are TRUE, 63% FALSE. When uncertain, lean FALSE.
- Implication is transitive: if A→B and B→C then A→C.
- Equations with MORE distinct variables are generally STRONGER (imply more).
- Variables on the RHS not appearing on LHS = "free variables" = MORE restrictive.

## PROCEDURE

### Step 1: Quick checks
- Equations identical? → TRUE
- Eq2 is "x = x"? → TRUE (tautology, everything implies it)
- Eq1 has form forcing all elements equal? → TRUE (implies everything)

### Step 2: Does Eq1 force a trivial operation?
Many equations force the operation to be projection (x*y = y or x*y = x) or constant.
If so, check if Eq2 holds for that operation.

KEY TECHNIQUE: Find a "left identity" element e (where e*x = x for all x).
If Eq1 lets you construct such an e for ANY starting elements, then Eq1 forces
x*y = y (right projection), and nearly all equations follow. Answer TRUE.

Example proof: If Eq1 is "x = ((y*y)*z)*x", fix y0,z0 and set e = (y0*y0)*z0.
Then e*x = x for all x, so e is a left identity. Substituting y=e in Eq1 and
simplifying shows x*y = y for all x,y. Then Eq2 follows trivially.

### Step 3: Try variable substitution
Can you instantiate Eq1's variables to derive Eq2?
- Look at Eq2's terms and try to "find" them inside Eq1's structure.
- You may need to apply Eq1 multiple times with different substitutions.
- If substitution works → TRUE.

### Step 4: Compare variable structure
- If Eq1 has more total variables than Eq2 → leans TRUE.
- If Eq2 has variables not constrained by Eq1 → leans FALSE.
- Eq2 with extra "free" universal quantifiers (like z,w that could be anything)
  is very hard to satisfy → likely FALSE unless Eq1 forces constant operation.

### Step 5: Try counterexample with 2-element magma
Check the magma {0,1} with these operation tables:
- x*y = 0 always (table: 00/00) — satisfies "constant" equations
- x*y = y (table: 01/01) — right projection, satisfies many equations
- x*y = x (table: 00/11) — left projection
- Table 01/10: x*y = 1-x (interesting edge cases)

For each: does it satisfy Eq1? If yes, does it satisfy Eq2?
If you find one satisfying Eq1 but not Eq2 → FALSE.

### Step 6: Default
If none of the above resolved it → FALSE (63% base rate).

## COMMON PATTERNS

### Always TRUE:
- Eq1 forces one-element magma (all elements equal) → implies everything
- Eq1 forces constant operation → implies most things
- Eq1 forces right/left projection → check if Eq2 holds for that projection
- Eq2 obtainable from Eq1 by substituting variables with terms

### Almost always FALSE:
- Eq2 has more free variables than Eq1
- Eq2 introduces different nesting structure not derivable from Eq1
- Eq1 is "weak" (many magmas satisfy it) but Eq2 is "strong" (few satisfy it)
