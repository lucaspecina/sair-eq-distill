# ALGORITHM: Does Eq1 imply Eq2 over all magmas?

Input: Eq1 (universally quantified equation), Eq2 (universally quantified equation)
Output: TRUE or FALSE

FACTS:
- 37% of pairs are TRUE, 63% FALSE
- Substitution is COMPLETE: if Eq1 implies Eq2, there is a finite chain of substitutions

PROCEDURE:

```
function CHECK_IMPLICATION(eq1, eq2):
    # Phase 1: Trivial cases
    if eq1 == eq2: return TRUE
    if eq2 == "x = x": return TRUE
    if eq1 can derive x = y: return TRUE  # implies everything

    # Phase 2: Variable analysis
    vars1 = count_distinct_variables(eq1)
    vars2 = count_distinct_variables(eq2)
    free_rhs1 = variables_in_rhs(eq1) - variables_in_lhs(eq1)
    free_rhs2 = variables_in_rhs(eq2) - variables_in_lhs(eq2)

    if vars2 > vars1 and len(free_rhs2) > len(free_rhs1):
        return FALSE  # eq2 has unconstrained variables

    # Phase 3: Try substitution proof (MOST IMPORTANT)
    # Try for at least 3 different substitution strategies

    # Strategy A: identify variables (set y:=x, z:=x, etc.)
    simplified = substitute_in_eq1(all_vars_equal_to_x)
    if simplified derives eq2: return TRUE

    # Strategy B: substitute terms from eq2 into eq1
    for each term t in eq2:
        result = substitute_in_eq1(variable := t)
        if result derives eq2: return TRUE

    # Strategy C: apply eq1 to itself
    result = apply_eq1_to_rhs_of_eq1()
    if result derives eq2: return TRUE

    # Phase 4: Check for forced operation types
    if eq1 forces constant operation (no shared vars between sides):
        return TRUE  # constant implies almost everything
    if eq1 forces right projection (can find universal left identity):
        if eq2 holds under right projection: return TRUE
    if eq1 forces left projection:
        if eq2 holds under left projection: return TRUE

    # Phase 5: Try counterexample
    for table in [{0,0,0,1}, {1,0,0,0}]:  # 2-element magmas
        if table satisfies eq1 and NOT eq2: return FALSE

    # Phase 6: Default
    return FALSE  # 63% base rate
```

CRITICAL WARNINGS:
- Do NOT skip Phase 3. Most errors come from saying FALSE without trying substitution.
- Substitution is COMPLETE by Birkhoff's theorem — if it's TRUE, substitution CAN prove it.
- Phase 5 (counterexample) should only be attempted AFTER Phase 3 fails.
